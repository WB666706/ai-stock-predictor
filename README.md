📈 AI 股票预测 Pro · A 股行情分析、智能诊断、深度学习预测与策略回测

一个基于 Streamlit + akshare + scikit-learn + LightGBM + PyTorch 的 A 股量化分析工具：输入股票代码，即可查看 K 线图与技术指标、获得技术面智能诊断评分、用机器学习 / 深度学习模型预测未来走势，将模型信号放入回测引擎检验真实表现，还能监控自选股并每日定时自动生成预测报告。

> ⚠️ 免责声明：本项目仅用于学习机器学习与量化分析技术，预测与回测结果不构成任何投资建议。股市有风险，入市需谨慎。

✨ 功能特性

🕯️ 行情与技术指标
 交互式 K 线图 + MA5/20/60 均线 + 成交量（红涨绿跌），可标注 MACD 金叉/死叉买卖点
 MACD、RSI、KDJ、布林带（BOLL）四大技术指标图

🧭 智能技术面诊断
 自动汇总 7 维信号：均线排列、MACD、RSI、KDJ、布林带位置、量能、20 日动量
 生成 0-100 综合评分仪表盘与逐项信号明细

🤖 AI 预测（9 种模型）
 专业模型：LightGBM（金融时序调参版）、时序安全 Stacking 堆叠融合（LightGBM + GBDT + RF，元特征由时序 out-of-fold 预测生成）
 经典模型：模型融合（Ensemble）、随机森林、梯度提升树、神经网络（MLP）、岭回归——直接多步预测，每个步长单独建模，避免递归误差累积
 深度学习：LSTM 与 Transformer（PyTorch）——以最近 30 日特征序列为输入，一次输出未来 1..N 日累计收益向量，早停选优
 约 40 维量价/形态/市场因子 + 时间序列交叉验证 / 时间留出验证（方向准确率、MAE、与基准对比）
 95% 置信区间、特征重要性、预测结果 CSV 导出
 滚动回放检验：模拟「每天只用当天之前的数据训练」，对比历史预测与实际走势

🎓 专业信号评估（量化机构标准）
 Purged 时序交叉验证：训练/测试集之间留出 embargo 空窗，消除标签重叠泄漏（参考 López de Prado《Advances in Financial Machine Learning》）
 IC / RankIC / ICIR：信息系数衡量预测与实际收益的相关性与稳定性
 分位数收益分析：按预测值分 5 组检验信号区分度（好信号应单调递增）
 方向概率模型：独立 LightGBM 分类器输出「次日上涨概率」与 AUC

🎯 智能选股（全市场两阶段漏斗）
 阶段一 · 快照粗筛（秒级）：全市场约 5000 只 → 剔除 ST/停牌/小市值/低流动性 → 按 60 日动量与资金活跃度保留前 200~500 只候选
 阶段二 · 横截面精排（约 1-3 分钟）：候选股 1.5 年日线 → 40 维因子面板 → 训练横截面 LightGBM（目标：未来 5 日收益，样本量是单股模型的数百倍）
 综合得分 = 50% AI 预测收益 + 25% 动量 + 25% 技术诊断，输出 Top 50/100 排名表，支持 CSV 导出
 数据获取专属提速通道：东财快照并行分页直连 + 腾讯 K 线单请求直连，多线程批量拉取

⚖️ 策略回测引擎
 模型信号策略（预测次日收益为正则持仓，否则空仓）vs 买入持有
 两种仓位方式：二元（满仓/空仓）与置信度加权（仓位与预测强度成正比，0~100%）
 滚动训练全程无未来数据泄漏，可调开仓阈值与交易成本
 输出总收益、年化收益、最大回撤、夏普比率、持仓胜率与仓位曲线

⭐ 自选股监控
 自选股增删管理（持久化到 `watchlist.json`）
 一键巡检：最新价、今日/5 日涨跌、诊断评分、AI 次日信号，支持 CSV 导出

⏰ 每日定时预测
 `daily_report.py` 自动巡检自选股，生成 Markdown + CSV 报告到 `reports/` 目录
 配合 Windows 计划任务即可每个交易日收盘后自动运行（见下文）

🆚 多股对比
 最多 6 只股票归一化走势对比、区间表现统计、日收益相关性热力图

📰 个股资讯
 最新新闻列表 + 关键词规则情绪标注（偏利好/中性/偏利空）

🇨🇳 数据源
 akshare 免费数据（无需注册和 API Key），东方财富为主、腾讯为备用，连接失败自动切换

🚀 快速开始

```bash
1. 克隆 / 下载本项目后，安装依赖
pip install r requirements.txt

2. 启动应用
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`，在左侧输入 6 位股票代码（如 `600519`）即可开始分析。

⏰ 设置每日定时预测（Windows）

```powershell
注册计划任务：每个交易日 15:30 自动巡检自选股并生成报告
schtasks /Create /TN "AI股票每日预测" /SC WEEKLY /D MON,TUE,WED,THU,FRI /ST 15:30 `
  /TR "cmd /c cd /d \"D:\path\to\aistockpredictor\" && python daily_report.py"

手动运行一次
python daily_report.py            巡检 watchlist.json 中的自选股
python daily_report.py 600519     巡检指定股票

取消任务
schtasks /Delete /TN "AI股票每日预测" /F
```

🧠 方法说明

1. 特征工程（约 40 维）：
   动量/均线：滞后收益率（1/2/3/5/10/20 日）、MA5/10/20/60 偏离度
   波动率：5/20 日滚动波动、短长波动比、ATR
   振荡指标：RSI、MACD 柱、KDJ、布林带位置、CCI、威廉 %R
   量能：量比、量能趋势、OBV 斜率、资金流量指标 MFI
   分布形态：20 日收益偏度/峰度、半年高低点位置
   K 线形态：振幅、跳空、实体、上下影线；日历效应（星期）
   大盘相对：上证指数 1/5/20 日收益、20 日相关性、60 日 Beta、20 日相对强度
2. 经典模型多步预测：预测未来第 h 个交易日时，以「未来 h 日累计收益率」为目标单独训练模型（h = 1…N），规避递归预测的误差累积。
3. LightGBM 采用针对金融低信噪比场景的保守参数（浅树 + 强正则 + 低学习率）；Stacking 的元特征由时序 out-of-fold 预测生成，元学习器（Ridge）学习基模型最优组合权重。
4. 深度模型序列预测：LSTM / Transformer 以 30 日特征序列为输入、未来 N 日收益向量为输出（多输出头），按时间留出最近 15% 样本做验证与早停。
5. 严格防泄漏：交叉验证使用 `TimeSeriesSplit`；专业评估使用带 embargo 空窗的 Purged CV；深度模型标准化仅用训练集统计量；滚动回放与策略回测均只用「当日之前」的数据训练。
6. 专业信号评估：IC（Pearson）、RankIC（Spearman）、ICIR（按折 IC 均值/标准差）、方向 AUC、分位数收益单调性——量化机构检验信号有效性的标准流程。
7. 置信区间：用留出样本的逐步长预测残差标准差估计 95% 置信区间。
8. 回测绩效：策略收益扣除换仓成本；支持二元与置信度加权两种仓位；输出年化收益、最大回撤、夏普比率、持仓日胜率等指标。
