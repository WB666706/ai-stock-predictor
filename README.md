# 📈 AI 股票预测 · A 股行情分析与机器学习走势预测

一个基于 **Streamlit + akshare + scikit-learn** 的 A 股分析与预测工具：输入股票代码，即可查看 K 线图、常用技术指标，并用机器学习模型预测未来走势（含置信区间与回测评估）。

> ⚠️ **免责声明**：本项目仅用于学习机器学习与量化分析技术，预测结果不构成任何投资建议。股市有风险，入市需谨慎。

## ✨ 功能特性

- 🕯️ **K 线图**：交互式蜡烛图 + MA5/20/60 均线 + 成交量（红涨绿跌）
- 📊 **技术指标**：MACD、RSI、KDJ、布林带（BOLL）
- 🤖 **AI 预测**：
  - 3 种可选模型：随机森林、梯度提升树、岭回归
  - **直接多步预测**：每个预测步长单独建模，避免递归误差累积
  - 19 个技术面特征：滞后收益率、均线偏离度、波动率、RSI、MACD、量能等
  - **时间序列交叉验证**回测：方向准确率（与基准对比）、收益率 MAE
  - 95% 置信区间与特征重要性可视化
- 🇨🇳 **免费数据源**：akshare（东方财富），无需注册和 API Key

## 🚀 快速开始

```bash
# 1. 克隆 / 下载本项目后，安装依赖
pip install -r requirements.txt

# 2. 启动应用
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`，在左侧输入 6 位股票代码（如 `600519`）即可开始分析。

## 🧠 预测方法说明

1. **特征工程**：从日线行情构造滞后收益率（1/2/3/5/10/20 日）、均线偏离度、滚动波动率、RSI、MACD 柱、KDJ、布林带位置、量比、振幅等特征。
2. **直接多步预测**：预测未来第 h 个交易日时，以「未来 h 日累计收益率」为目标单独训练模型（h = 1…N），规避递归预测的误差累积。
3. **回测评估**：使用 `TimeSeriesSplit` 时间序列交叉验证（杜绝未来数据泄漏），报告次日涨跌方向准确率，并与「总是预测上涨」的基准对比。
4. **置信区间**：用最近 20% 留出样本的预测残差标准差估计 95% 置信区间。
