📈 AI 股票预测 Pro · A 股行情分析、智能诊断、深度学习预测与策略回测

一个基于 Streamlit + akshare + scikitlearn + PyTorch 的 A 股量化分析工具：输入股票代码，即可查看 K 线图与技术指标、获得技术面智能诊断评分、用机器学习 / 深度学习模型预测未来走势，将模型信号放入回测引擎检验真实表现，还能监控自选股并每日定时自动生成预测报告。

> ⚠️ 免责声明：本项目仅用于学习机器学习与量化分析技术，预测与回测结果不构成任何投资建议。股市有风险，入市需谨慎。

✨ 功能特性

🕯️ 行情与技术指标
 交互式 K 线图 + MA5/20/60 均线 + 成交量（红涨绿跌），可标注 MACD 金叉/死叉买卖点
 MACD、RSI、KDJ、布林带（BOLL）四大技术指标图

🧭 智能技术面诊断
 自动汇总 7 维信号：均线排列、MACD、RSI、KDJ、布林带位置、量能、20 日动量
 生成 0-100 综合评分仪表盘与逐项信号明细

🤖 AI 预测（7 种模型）
 经典模型：模型融合（Ensemble）、随机森林、梯度提升树、神经网络（MLP）、岭回归——直接多步预测，每个步长单独建模，避免递归误差累积
 深度学习：LSTM 与 Transformer（PyTorch）——以最近 30 日特征序列为输入，一次输出未来 1..N 日累计收益向量，早停选优
 19 个技术面特征 + 时间序列交叉验证 / 时间留出验证（方向准确率、MAE、与基准对比）
 95% 置信区间、特征重要性、预测结果 CSV 导出
 滚动回放检验：模拟「每天只用当天之前的数据训练」，对比历史预测与实际走势

⚖️ 策略回测引擎
 模型信号策略（预测次日收益为正则持仓，否则空仓）vs 买入持有
 滚动训练全程无未来数据泄漏，可调开仓阈值与交易成本
 输出总收益、年化收益、最大回撤、夏普比率、持仓胜率与仓位曲线

⭐ 自选股监控
 自选股增删管理（持久化到 `watchlist.json`）
 一键巡检：最新价、今日/5 日涨跌、诊断评分、AI 次日信号，支持 CSV 导出

⏰ 每日定时预测
 `daily_report.py` 自动巡检自选股，生成 Markdown + CSV 报告到 `reports/` 目录
 配合 Windows 计划任务即可每个交易日收盘后自动运行（见下文）

🆚 多股对比
 最多 6 只股票归一化走势对比、区间表现统计、日收益相关性热力图

📰 个股资讯
 最新新闻列表 + 关键词规则情绪标注（偏利好/中性/偏利空）

🇨🇳 数据源
 akshare 免费数据（无需注册和 API Key），东方财富为主、腾讯为备用，连接失败自动切换

🚀 快速开始

```bash
1. 克隆 / 下载本项目后，安装依赖
pip install r requirements.txt

2. 启动应用
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`，在左侧输入 6 位股票代码（如 `600519`）即可开始分析。

⏰ 设置每日定时预测（Windows）

```powershell
注册计划任务：每个交易日 15:30 自动巡检自选股并生成报告
schtasks /Create /TN "AI股票每日预测" /SC WEEKLY /D MON,TUE,WED,THU,FRI /ST 15:30 `
  /TR "cmd /c cd /d \"D:\path\to\aistockpredictor\" && python daily_report.py"

手动运行一次
python daily_report.py            巡检 watchlist.json 中的自选股
python daily_report.py 600519     巡检指定股票

取消任务
schtasks /Delete /TN "AI股票每日预测" /F
```

🧠 方法说明

1. 特征工程：滞后收益率（1/2/3/5/10/20 日）、均线偏离度、滚动波动率、RSI、MACD 柱、KDJ、布林带位置、量比、振幅等 19 个特征。
2. 经典模型多步预测：预测未来第 h 个交易日时，以「未来 h 日累计收益率」为目标单独训练模型（h = 1…N），规避递归预测的误差累积。
3. 深度模型序列预测：LSTM / Transformer 以 30 日特征序列为输入、未来 N 日收益向量为输出（多输出头），按时间留出最近 15% 样本做验证与早停。
4. 严格防泄漏：交叉验证使用 `TimeSeriesSplit`；深度模型标准化仅用训练集统计量；滚动回放与策略回测均只用「当日之前」的数据训练。
5. 置信区间：用留出样本的逐步长预测残差标准差估计 95% 置信区间。
6. 回测绩效：策略收益扣除换仓成本；输出年化收益、最大回撤、夏普比率、持仓日胜率等指标。
